-----READ ME-----

Thank you for downloading this package!

This package is a game-ready grenade system you can plug-and-play into your own project, use to learn how
to create your own grenade system, or fully customise.

This package contains one Grenade - the Hi-Explosive grenade. 
The full life-cycle of using this asset in-game has been crafted for you to use, including
Pickups, throwing, explosion effect, damage handling, sound effects, UI, and physics effects.

This package also includes one test scene with a player object you can use right away, with it's own first-person controller
ready to go for testing!

The folders and resources included in this package are:

-Audio
-Materials
-Meshes 
-Prefabs
-Scenes
-Scripts
-Textures
-lighting

Explore these folders in the inspector to get a full inventory of what the package contains.

The scripts are fully commented, with inspector-headings so you can cusomise the effects and mechanics of the grenade and throwing system
all in the unity editor without touching code if you'd prefer.

The Scripts and their function are as follows:

- Damage Handling
This script handles when the Player or Enemy is damaged by the blast of the grenade. This is the 'Health' System where damage is applied

- Grenade Pickup
This script is attatched to the Grenade Pickup prefab, and allows the player to pick up new grenades when colliding with it.

- Grenade System
This script handles the Grenade inventory of the player, as well as the player input when throwing the grenade and throw force, 

- HiEx Grenade
This script is attatched to the thrown grenade, and handles the delay timer, damage, audio and explosion effects, as well as the physics of the explosion.
There are three damage radii,or distances - close, near, and far, and a player or enemy wil take less damage the further away from the explosion they are.
All settings are fully customisable.

- Player Controller
This is a simple player controller attatched to the player object included in the Test scene. You can of course attatch the grenade system to your own player controller,
as this script is just for testing purposes.
